Requests for Students
1) In order to get student with id(withallcourses) ==>http://localhost:8080/student/id=? (GET)
2) In order to get all students (withallcourses) ==>http://localhost:8080/student/list  (GET)
3) In order to add or update  ==>http://localhost:8080/student/add-update  (POST)


Requests for Courses
1) In order to get course with ID (with all students enrolled) http://localhost:8080/course/id=? (GET)
2) In order to get all courses (with all students)  http://localhost:8080/course/list (GET)
